# x402 AI Agent Demo テスト実施手順

---

## 環境セットアップ（初回のみ）

### 1. Node.js の確認

```bash
node -v   # 18以上であればOK
```

---

### 2. リポジトリのクローン

```bash
git clone https://github.com/HlDEy/x402-demo-2026-0406.git
cd x402-demo-2026-0406
```

---

### 3. `.env` ファイルの作成・記入

**Provider側：**

```bash
cp provider/.env.example provider/.env
```

`provider/.env` を開いて以下を記入：

| 変数 | 内容 |
|------|------|
| `EVM_ADDRESS` | 受取ウォレットのアドレス（0x...） |

**Agent側：**

```bash
cp agent/.env.example agent/.env
```

`agent/.env` を開いて以下を記入：

| 変数 | 内容 |
|------|------|
| `WALLET_PRIVATE_KEY` | 支払いウォレットの秘密鍵（Base Sepolia専用） |
| `ANTHROPIC_API_KEY` | https://console.anthropic.com で取得 |

---

### 4. テスト用トークンの取得（初回のみ）

`WALLET_PRIVATE_KEY` のアドレスに対して：

1. テストETH → https://www.alchemy.com/faucets/base-sepolia
2. テストUSDC → [https://faucet.circle.com](https://faucet.circle.com)（Base Sepoliaを選択）

---

### 5. サーバーの起動

```bash
bash start.sh
```

3つのターミナルが自動で開き、ブラウザが起動します。  
初回実行時は `.env` の入力プロンプトが表示されます。

---

## 事前確認

- [ ] Provider Server が起動している（`http://localhost:3001`）
- [ ] Agent Server が起動している（`http://localhost:3002`）
- [ ] Frontend が起動している（`http://localhost:5173`）
- [ ] `agent/.env` に `WALLET_PRIVATE_KEY` と `ANTHROPIC_API_KEY` が記入済み
- [ ] `provider/.env` に `EVM_ADDRESS` が記入済み
- [ ] 支払いウォレットにテストETH・テストUSDCが入っている

---

## テスト手順

### 1. ブラウザで開く

```
http://localhost:5173
```

---

### 2. 入力フォームに値を入力

| 項目 | 内容 |
|------|------|
| Location | 地域名（例：Tokyo） |
| Urgency | Low / Medium / High / Critical から選択 |
| 実際の価格 | バックエンド設定値（読み取り専用） |
| Budget | ウォレット残高（読み取り専用） |

---

### 3. 「Run Agent」ボタンをクリック

---

### 4. 結果を確認

**確認ポイント：**

| 確認項目 | 場所 |
|---------|------|
| 無料天気が表示されるか | 結果 > 天気情報（Free） |
| Claudeの判断が表示されるか | 結果 > Claudeの判断 |
| 判断理由が表示されるか | 結果 > Claudeの判断 |
| 支払い状況が表示されるか | 結果 > 決済 |
| プレミアムデータが表示されるか（BUYの場合） | 結果 > プレミアムデータ |
| ログが流れるか | Agent Log |

---

### 5. ログの流れを確認

BUYの場合、以下の順でログが流れることを確認：

```
[Agent] Fetching price from Provider...
[Provider] Price: 0.01 USDC
[Agent] Fetching free weather from Provider...
[Provider] Weather in Tokyo: clear
[Agent] Evaluating premium data (urgency=high, price=0.01 USDC, budget=...)
[Claude] Decision: BUY
[Claude] Reason: ...
[Agent] Requesting premium endpoint from Provider...
[Provider] 402 Payment Required
[Agent] Executing autonomous payment via x402...
[Agent] Payment confirmed. Premium data received from Provider.
```

SKIPの場合：

```
[Agent] Fetching price from Provider...
[Provider] Price: 0.01 USDC
[Agent] Fetching free weather from Provider...
[Provider] Weather in Tokyo: clear
[Agent] Evaluating premium data (urgency=low, price=0.01 USDC, budget=...)
[Claude] Decision: SKIP
[Claude] Reason: ...
```

---

## エラー時の確認手順

1. ブラウザの開発者ツール（F12）→ Consoleタブを確認
2. 各ターミナル（Provider / Agent）にエラーログが出ていないか確認
3. `.env` の値が正しく記入されているか再確認
4. テストUSDCの残高が不足していないか確認
